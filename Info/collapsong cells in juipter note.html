The Registry Editor in Windows contains system settings that control how your operating system behaves. Adjusting scale factors in the Registry can help Windows retain monitor settings across restarts, particularly when dealing with DPI (dots per inch) scaling issues on high-resolution monitors.

Here’s how to approach adjusting LogPixels in the Registry Editor for DPI scaling:

What is LogPixels?

  • LogPixels is a registry key that controls DPI scaling, which adjusts how large text, icons, and other UI elements appear on your screen.
  • Changing the LogPixels value can help Windows maintain the correct scale factor for your monitors, especially if it resets every time you start up.

How to Edit LogPixels in the Registry:

  1. Open Registry Editor:

    • Press Win + R, type regedit, and press Enter. This opens the Registry Editor.

  2. Navigate to the LogPixels Key:

    • Go to HKEY_CURRENT_USER\Control Panel\Desktop.
    • Find the LogPixels entry on the right side of the window. If it’s not there, it may be listed as Win8DpiScaling or DpiValue depending on your version of Windows.

  3. Adjust LogPixels Value:

    • Double-click on LogPixels to edit its value. Common values are:
      • 96 for 100% scaling (default),
      • 120 for 125% scaling,
      • 144 for 150% scaling,
      • 192 for 200% scaling.
    • Set the value to match your preferred scaling factor. For instance, setting it to 120 will make elements appear larger, similar to 125% scaling.

  4. Restart Your Computer:

    • After setting the desired value, close the Registry Editor and restart your computer to apply the changes.

Important Notes:

  • Back Up Your Registry: Before making any changes, back up your registry by selecting File > Export and saving a copy. This way, you can restore settings if anything goes wrong.
  • Be Cautious: Editing the registry can affect your system, so only modify values you are familiar with.

  3. Driver Issues: If your graphics driver is outdated or incompatible, scaling and display settings may reset.
  4. Corrupt User Profile: In rare cases, a corrupt user profile can prevent changes from being saved.

Fixes to Prevent Settings from Resetting

Here’s how you can ensure your configurations are saved automatically:

  1. Turn Off Fast Startup:

    • Go to Control Panel > Power Options > Choose what the power buttons do.
    • Click Change settings that are currently unavailable, then uncheck Turn on fast startup.
    • Restart your computer and adjust your settings to see if they’re retained.

  2. Run Windows as Administrator:

    • Right-click the Start menu, open Windows PowerShell (Admin) or Command Prompt (Admin).
    • Run the command sfc /scannow to check for system file issues that could prevent settings from saving.

  3. Update Graphics Driver:

    • Make sure your display drivers are up-to-date, as outdated or incompatible drivers can reset display settings.

  4. Set Persistent Registry Entries for Scale Factors:

    • Open Registry Editor and navigate to HKEY_CURRENT_USER\Control Panel\Desktop.

Automatic Reset of Temporary Configurations

If you’re frequently resetting settings (e.g., connecting/disconnecting monitors), Windows might recognize the configuration as temporary and may not save it reliably.
